# 📖 TechBright <a name="about-project"></a>

**TechBright** is a blogging website created using ruby on rails. It includes user registration, login, create posts, like or comment on the posts. See posts by users, see likes counts, comment counts and more.

## 🛠 Built With <a name="built-with"></a>
